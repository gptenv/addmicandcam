# Safety and Consent

LLM Telepresence Browser Lab is for authorized testing, demonstrations, accessibility experiments, and user-owned or permissioned video-chat environments.

Use it only where you have permission to participate with the account, site, meeting, room, or conversation being tested. Do not use it to mislead people about synthetic media, evade platform controls, bypass verification, collect credentials, bypass CAPTCHAs, or abuse services.

The default generated-camera path supports a visible disclosure watermark. Keep this enabled for demos and any environment where other participants should know that media is generated or AI-assisted.

The project intentionally does not include stealth, anti-detection bypass, CAPTCHA bypass, credential theft, or impersonation features.
